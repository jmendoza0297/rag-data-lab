¿Qué Motor de Extracción elegir?

  • PyMuPDF Híbrido: Tu comodín. Si el PDF tiene texto lo saca rápido, si tiene imágenes borrosas les pasa OCR automáticamente.
  • pdfplumber: Úsalo SÓLO si tu PDF parece una hoja de cálculo o tiene formatos de columnas complejos.
✂️

El Peligro de los Fragmentos Pequeños

Si cortas el texto a 200 caracteres, vas a partir oraciones y títulos largos por la mitad (ej: "Universidad Nacional..." | "...de Ingeniería").

Recomendación: Usa 1000 a 1500 caracteres para dar párrafos enteros de contexto a la IA.

🧩 ¿Cómo funciona el Solapamiento (Overlap)?

Imagina que el sistema corta un texto justo entre estas dos frases: "Juan compró un coche rojo." y "Este vehículo le costó caro."
Si la IA lee solo el segundo fragmento, no sabrá qué es "Este vehículo". El Solapamiento (15%) hace que el final del fragmento 1 se repita en el fragmento 2, pegando las ideas lógicas para que Qdrant las entienda juntas.

📄 Chunk 1:
Juan compró un coche rojo. Este vehículo
📄 Chunk 2 (con Solapamiento):
Este vehículo le costó muy caro.
